GetString(...)
getstring([cronly] [, prompt])  ->  float or keyword
 
Prompts the user for an string interactively and returns that string.
The prompt argument is a string and is printed to the console.
On R14 or earlier, the returned string is truncated to 132 characters.
If the integer cronly argument exists and is nonzero, the string can
contain spaces and the user must terminate it by entering <Return>.
If cronly is emitted or zero, entering either a space or <Return>;
terminates the string.
 
If a call to initget() before the getreal() specified keywords,
then one of those keywords can be returned. Flags to initget() are
respected where appropriate.
GetVar(...)
getvar(varname)  ->  Value
 
Get the value of an Autocad system variable (varname is a string).
If the variable varname doesn't exist, or if the value can't be
converted to a Python datatype, a ValueError exception is raised.

SetVar(...)
setvar(varname, value)
 
Set the value of an Autocad system variable.
 
If the variable varname doesn't exist, if the value can't
be converted to an Autolisp datatype, if the value doesn't
match the type or range of the variable or if the variable
is read-only, a ValueError exception is raised.
